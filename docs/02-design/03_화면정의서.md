# 화면 정의서

## 1. 문서 정보

* **문서 목적**: 동네책장 플랫폼의 화면 구성과 흐름을 정의하여 UI/UX 설계를 명확히 한다.
* **대상**: 기획, 디자인, 프론트엔드
* **작성일**: 2026-01-20
* **버전**: 1.0

---

## 1. 화면 목록

|---|---|---|
| 로그인 | 인증 및 진입 | 이메일, 비밀번호 입력, 로그인 유지 체크박스, 로그인 버튼 |
| 회원가입 | 계정 생성 | 이메일, 비밀번호, 확인, 닉네임, 가입 버튼 |
| 메인(책 목록) | 책 탐색 | 책 목록, 검색 |

| 책 등록/수정 | 책 등록 및 편집 | 제목, 저자, 카테고리, 상태, 설명, 이미지 업로드 버튼 |
| 책 검색 | 책 찾기 | 검색어 입력, 검색 결과 목록 |
| 찜 목록 | 관심 책 관리 | 찜한 책 목록 |
| 채팅 목록 | 채팅방 확인 | 채팅방 목록, 상대방 닉네임, 마지막 메시지, 시간 |
| 채팅방 | 실시간 채팅 | 메시지 목록, 메시지 입력창, 전송 버튼 |
| 모임 게시판 목록 | 모임 탐색 | 모임 글 목록, 검색, 글쓰기 버튼 |
| 모임 글 상세 | 모임 정보 확인 | 제목, 내용, 일시, 장소, 댓글 목록, 댓글 입력 |
| 모임 글 작성/수정 | 모임 글 등록 및 편집 | 제목, 내용, 일시, 장소 입력, 저장 버튼 |

| 프로필 수정 | 정보 수정 | 닉네임, 프로필 이미지 입력, 저장 버튼 |
| 알림 | 알림 확인 | 알림 목록 (댓글 알림, 채팅 알림) |

## 2. 화면 흐름

### 2.1 비회원 흐름
* 메인(책 목록) → 책 상세 조회
* 메인(책 목록) → 책 검색 → 검색 결과 → 책 상세
* 모임 게시판 목록 → 모임 글 상세 조회
* 로그인 필요 기능 접근 시 → 로그인 화면 이동

### 2.2 회원 흐름 - 책 거래
* 로그인 → 메인(책 목록) → 책 상세 → 찜하기
* 로그인 → 메인(책 목록) → 책 등록 → 등록 완료 → 책 상세
* 로그인 → 메인(책 목록) → 책 상세 → 수정/삭제 (본인 글만)
* 로그인 → 찜 목록 → 책 상세

### 2.3 회원 흐름 - 채팅
* 로그인 → 책 상세 → 채팅하기 버튼 → 채팅방
* 로그인 → 채팅 목록 → 채팅방 선택 → 실시간 채팅
* 로그인 → 채팅방 → 메시지 전송 및 수신

### 2.4 회원 흐름 - 모임
* 로그인 → 모임 게시판 목록 → 모임 글 작성 → 모임 글 상세
* 로그인 → 모임 게시판 목록 → 모임 글 상세 → 댓글 작성
* 로그인 → 모임 게시판 목록 → 모임 글 상세 → 수정/삭제 (본인 글만)

### 2.5 회원 흐름 - 후기 및 프로필
* 로그인 → 프로필 조회 → 프로필 수정
* 로그인 → 프로필 조회 → 후기 작성
* 로그인 → 프로필 조회 → 작성한 후기 / 받은 후기 조회

## 3. 화면별 상세

### 3.1 로그인
* **입력 항목**: 이메일, 비밀번호, 로그인 유지 체크박스
* **행동**: 로그인, 회원가입 이동
* **상태**: 에러 메시지 (계정 없음/비밀번호 불일치)

### 3.2 회원가입
* **입력 항목**: 이메일, 비밀번호, 비밀번호 확인, 닉네임
* **행동**: 가입, 로그인 이동
* **상태**: 중복 이메일, 비밀번호 규칙 위반, 닉네임 중복

### 3.3 메인(책 목록)
* **구성**: 검색바(카테고리 선택), 책 목록, 무한 스크롤
* **행동**: 검색, 책 등록 이동, 책 상세 이동
* **상태**: 결과 없음 안내, 로딩 상태

### 3.4 책 상세
* **구성**: 책 이미지, 제목, 저자, 상태, 설명, 판매자 정보, 찜하기 버튼, 채팅하기 버튼, 수정/삭제 버튼(본인만)
* **행동**: 찜하기/찜 해제, 채팅하기(판매자와 채팅 시작), 수정 이동(본인만), 삭제(본인만), 판매자 프로필 이동
* **상태**: 찜 상태 표시, 권한 없음 안내

### 3.5 책 등록/수정
* **구성**: 제목 입력, 저자 입력, 카테고리 선택, 상태 선택, 설명 입력, 이미지 업로드, 저장/취소 버튼
* **행동**: 저장 시 책 상세 이동, 취소 시 이전 화면
* **상태**: 필수 항목 누락 안내, 이미지 크기 초과 안내

### 3.6 책 검색
* **구성**: 검색어 입력, 검색 결과 목록, 필터 옵션
* **행동**: 검색 실행, 책 상세 이동
* **상태**: 검색 결과 없음 안내

### 3.7 찜 목록
* **구성**: 찜한 책 목록
* **행동**: 찜 해제, 책 상세 이동
* **상태**: 찜한 책 없음 안내

### 3.8 채팅 목록
* **구성**: 채팅방 목록, 상대방 닉네임, 상대방 프로필 이미지, 마지막 메시지, 시간, 안 읽은 메시지 개수
* **행동**: 채팅방 선택, 채팅방 삭제
* **상태**: 채팅 없음 안내, 로딩 상태

### 3.9 채팅방
* **구성**: 상대방 정보(닉네임, 프로필 이미지), 메시지 목록(시간, 내용, 읽음 표시), 메시지 입력창, 전송 버튼
* **행동**: 메시지 입력 및 전송, 실시간 메시지 수신, 이전 메시지 스크롤 로드
* **상태**: 연결 끊김 안내, 메시지 전송 실패 안내

### 3.10 모임 게시판 목록
* **구성**: 검색바, 모임 글 목록(제목, 작성자, 일시, 작성일), 글쓰기 버튼, 무한 스크롤
* **행동**: 검색, 글쓰기 이동, 모임 글 상세 이동
* **상태**: 결과 없음 안내, 로그인 필요 안내(글쓰기 시)

### 3.11 모임 글 상세
* **구성**: 제목, 내용, 일시, 장소, 작성자/작성일, 댓글 목록, 댓글 입력창, 수정/삭제 버튼(본인만)
* **행동**: 댓글 작성/삭제, 글 수정/삭제(본인만)
* **상태**: 댓글 없음 안내, 권한 없음 안내

### 3.12 모임 글 작성/수정
* **구성**: 제목 입력, 내용 입력, 일시 선택, 장소 입력, 저장/취소 버튼
* **행동**: 저장 시 모임 글 상세 이동, 취소 시 이전 화면
* **상태**: 필수 항목 누락 안내

### 3.13 후기 작성
* **구성**: 거래 상대 정보, 평점 선택(별점), 내용 입력, 등록 버튼
* **행동**: 등록 시 프로필 페이지 이동
* **상태**: 거래 내역 없음 안내, 내용 누락 안내

### 3.14 프로필 조회
* **구성**: 프로필 이미지, 닉네임, 등록 책 수, 작성한 후기 탭, 받은 후기 탭, 프로필 수정 버튼(본인만)
* **행동**: 작성한 후기/받은 후기 탭 전환, 프로필 수정 이동(본인만), 등록한 책 목록 조회
* **상태**: 후기 없음 안내

### 3.15 프로필 수정
* **구성**: 닉네임 입력, 프로필 이미지 업로드, 저장/취소 버튼
* **행동**: 저장 시 프로필 조회 이동, 취소 시 이전 화면
* **상태**: 닉네임 중복 안내

### 3.16 알림
* **구성**: 알림 목록 (찜한 책 변동, 댓글 알림, 채팅 알림 등), 알림 삭제 버튼
* **행동**: 알림 클릭 시 해당 페이지 이동, 알림 삭제
* **상태**: 알림 없음 안내

## 4. 공통 UI 요소

* **헤더**: 로고, 검색바, 로그인/프로필 아이콘, 알림 아이콘
* **네비게이션**: 홈(책 목록), 채팅, 모임, 찜 목록, 내 프로필
* **푸터**: 서비스 소개, 개인정보처리방침, 문의하기

## 5. 참고사항

* 모든 화면은 모바일 우선 반응형 디자인으로 제공된다.
* Figma 디자인 시안을 기준으로 개발한다.
* 로딩 상태와 에러 상태는 모든 화면에서 일관성 있게 표시한다.
* 모달 방식의 로그인/회원가입 적용 가능.
* 실시간 채팅은 WebSocket 또는 Socket.io를 사용한다.
